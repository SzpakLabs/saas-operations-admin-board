# Portfolio Notes — OpsBoard

## Project Title

**OpsBoard — SaaS Operations Admin Dashboard (Nuxt 3 + TypeScript)**

## Upwork Description

> Full-stack admin panel built with Nuxt 3, Nuxt UI v4, Pinia, and Chart.js. Demonstrates production-grade internal tooling for SaaS operations teams: customer management, subscription tracking, support ticket workflows, operational alerts, and an analytics dashboard with live charts. Features working search, sort, filters, paginated tables, split-panel detail views, local state mutations, and dark mode support. Clean TypeScript codebase with realistic mock data — no backend required, fully deployable to Vercel.

## Suggested Screenshots

1. **Dashboard Overview** — Full KPI card row + MRR line chart + bar chart side by side + alerts panel. Best screenshot to lead with.
2. **Customers Table** — Filtered table showing status badges, health score progress bars, plan badges, sorted columns.
3. **Customer Detail View** — Full customer profile sidebar + subscription card + invoice table + internal notes.
4. **Tickets Split Panel** — Left list with priority badges selected + right detail panel with status buttons visible.
5. **Alerts Page** — Mix of Critical/High/Medium alert cards with the severity left-border accents + action buttons (Investigate, Resolve, Dismiss).
